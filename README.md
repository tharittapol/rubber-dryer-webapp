# Rubber Dryer Web (Next.js)

Web project outline (Desktop + Mobile responsive)

## Requirements
- Node.js 18+ (recommend 20+)

## Run
```bash
npm install
npm run dev
```

open http://localhost:3000

## Routes
- /login
- /dashboard
- /control
- /settings/profiles
- /settings/factories
- /settings/rooms
- /settings/users
